Quem somos

“Cotação de Livros ML” é uma aplicação de análise de mercado que consulta ofertas públicas de livros no Mercado Livre para fins estatísticos e comparativos.

Dados que tratamos

  • Dados públicos de anúncios (título, preço, URL, atributos do produto) obtidos via API do Mercado Livre.
  • Não coletamos nem armazenamos dados pessoais de consumidores finais.

Base legal

O tratamento se baseia em legítimo interesse para análise de mercado e em conformidade com os Termos da API do Mercado Livre.

Retenção

Conservamos apenas agregações estatísticas (como médias de preço). Logs técnicos são mantidos por até 90 dias para auditoria.

Compartilhamento

Não compartilhamos dados com terceiros, exceto quando exigido por lei.

Segurança

As chaves e tokens de acesso são protegidos e não são expostos publicamente. O acesso é restrito.

Seus direitos
